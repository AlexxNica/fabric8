Open Source developer platform for Kubernetes and OpenShift

Continuous made Easy

Fabric8 makes it easy to develop microservices using automatic Continuous Integration and Continuous Delivery

Get Started

Fabric8 is available in two different packages depending on your requirements

Fabric8 Jenkins

  • Jenkins and Nexus with fabric8 CI / CD features enabled
  • Simple to install in any namespace on Kubernetes/OpenShift
  • Delegates to OpenShift and/or GitHub for authentication

Fabric8 Platform

  • multi-tenant CI / CD platform with KeyCloak for SSO
  • each user/team gets their own Jenkins
  • unified web console with wizards, runtime views, IDE, planning and analytics
